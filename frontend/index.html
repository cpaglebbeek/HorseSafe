HorseSafe

Jouw kluis. Niet de onze.

Zero-knowledge wachtwoord-vault

HorseSafe slaat je wachtwoorden veilig op met een KDBX4-versleutelde kluis. Het master-wachtwoord en de keyfile verlaten nooit je browser — de server ziet enkel ciphertext. Bij verlies van je master-wachtwoord is je data permanent weg.

Compatibel met KeePassXC desktop voor disaster-recovery. Open-source-overweging vanaf v0.1.0. Hier is de POC — v0.0.2-Hellman.
